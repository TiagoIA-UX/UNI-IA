## Por que animais ameaçados?

Não é estética — é compromisso.

Cada nome no sistema é uma espécie sob pressão de extinção. O **Dízimo Amazônia** — 10% do lucro líquido operacional — é destinado a organizações de conservação dessas espécies. Quando o sistema performa, a fauna performa junto.

> "Estratégia sem governança é aposta. Preservação sem financiamento é intenção."

---

## Regras de nomenclatura

- O **ID interno** (`AEGIS`, `ATLAS`, etc.) nunca muda — é usado pela API e pelo banco de dados
- O **nome do guardião** é o nome exibido na interface e na documentação pública
- O **estado** é a referência oficial da ave/animal símbolo (fonte: IBGE / legislação estadual)
- Novos agentes seguem o mesmo padrão: animal símbolo de estado + função clara

## Fluxo de decisão

```
8 guardiões votam
        ↓
   Boitatá sintetiza
   (score 0–100, direção, classificação)
        ↓
   Jabuti valida risco
   (gates: score, whitelist, drawdown, aprovação manual)
        ↓
   Ordem executada ou vetada
        ↓
   Peixe-Boi monitora outcome
   (feedback para o próximo ciclo)
```
